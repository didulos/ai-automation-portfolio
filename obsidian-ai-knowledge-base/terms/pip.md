# pip

## 정의

pip는 파이썬(Python) 패키지를 설치하고 관리하는 패키지 관리자(package manager)이다. 파이썬 커뮤니티에서 만든 라이브러리와 도구들을 중앙 저장소([[PyPI]])에서 다운로드하여 로컬 환경에 설치할 수 있게 해준다. 대부분의 현대적인 파이썬 프로젝트에서 필수적인 도구이다.

## 왜 중요한가

머신러닝 및 데이터 과학 프로젝트에서 [[NumPy]], [[Pandas]], [[TensorFlow]] 등 수백 개의 외부 라이브러리를 효율적으로 관리해야 하기 때문에, pip는 개발 환경 구축의 가장 기본적인 필수 도구이다. 프로젝트의 의존성(dependency) 관리를 통해 재현 가능한 개발 환경을 유지할 수 있다.

## 비유로 이해하기

pip는 마치 온라인 쇼핑몰에서 물건을 주문하고 배송받는 것과 같다. 설치하고 싶은 라이브러리의 이름을 명령어로 입력하면, 중앙 창고([[PyPI]])에서 해당 패키지를 찾아 자동으로 다운로드하고 설치해주는 방식이다.

## 실제 사용 예시

```bash
pip install numpy pandas scikit-learn
pip install -r requirements.txt
```

머신러닝 모델 개발에 필요한 주요 라이브러리들을 한 줄의 명령어로 일괄 설치하거나, 프로젝트의 모든 의존성을 `requirements.txt` 파일에 명시하여 동일한 환경을 재현할 수 있다.

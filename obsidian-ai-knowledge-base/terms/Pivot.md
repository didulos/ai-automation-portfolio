# Pivot

## 정의

Pivot은 데이터의 행(row)과 열(column)을 회전시켜 데이터 구조를 재구성하는 작업이다. 원본 데이터의 특정 값을 인덱스로 설정하고, 다른 열의 값들을 새로운 열로 변환하여 데이터를 요약하거나 분석하기 쉬운 형태로 변환한다.

## 왜 중요한가

데이터 분석(data analysis) 및 전처리(preprocessing) 단계에서 원하는 관점으로 데이터를 재구성할 수 있어, 패턴 발견과 의사결정을 훨씬 효율적으로 만든다. 특히 시계열 데이터나 다차원 데이터를 다룰 때 필수적인 기법이다.

## 비유로 이해하기

학생들의 시험 성적표를 생각해보자. 원본은 "학생명, 시험명, 점수" 형태의 세로 리스트인데, Pivot을 사용하면 "학생명을 행, 시험명을 열"로 변환하여 각 학생의 전체 성적을 한눈에 볼 수 있는 표로 만들 수 있다.

## 실제 사용 예시

```python
# Pandas에서 Pivot 사용
df.pivot(index='학생', columns='시험과목', values='점수')
# 결과: 학생을 행으로, 시험과목을 열로 하는 점수표 생성
```

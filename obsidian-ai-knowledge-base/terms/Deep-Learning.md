# Deep Learning

## 정의
심층 신경망(deep neural network)을 이용하여 대량의 데이터에서 패턴을 학습하는 [[AI-Terms/01-Terms/Machine-Learning]](머신러닝)의 한 분야이다. 여러 겹의 [[Hidden-Layer]](은닉층)을 통해 데이터의 고수준 특징(high-level feature)을 자동으로 추출하고 학습한다.

## 왜 중요한가
이미지 인식, 자연어 처리, 음성 인식 등 복잡한 실제 문제 해결에 가장 효과적인 방법론이며, 현대 [[AI]](인공지능) 시스템의 핵심 기술이다.

## 관련 개념
- **상위 개념:** [[AI-Terms/01-Terms/Machine-Learning]](머신러닝)
- **하위 개념:** 
  - [[CNN]](합성곱 신경망)
  - [[RNN]](순환 신경망)
- **연관 개념:** 
  - [[Neural-Network]](신경망)
  - [[Backpropagation]](역전파)

## 비유로 이해하기
마치 학생이 기초 과목부터 시작해 점진적으로 더 어려운 과목을 학습하며 전문성을 쌓는 것처럼, 심층 신경망은 낮은 층에서 단순한 패턴(선, 색)을 학습하고 높은 층으로 갈수록 복잡한 개념(얼굴, 물체)을 인식하게 된다.

## 실제 사용 예시
ChatGPT와 같은 [[LLM]](대형언어모델)은 수십억 개의 파라미터를 가진 심층 신경망으로 구성되어 있으며, 의료 진단 AI는 심층 신경망을 통해 X-ray 이미지에서 질병의 징후를 자동으로 감지한다.

# ETL

## 정의

[[ETL]](ETL)은 데이터를 추출(Extract), 변환(Transform), 적재(Load)하는 일련의 데이터 파이프라인 프로세스이다. 여러 소스에서 수집한 원본 데이터를 정제하고 가공한 후 목표 시스템에 저장하는 데이터 통합(data integration) 방식이다.

## 왜 중요한가

기업이 보유한 다양한 데이터 소스들을 통합하여 일관된 형태의 데이터를 만들어야 분석과 의사결정이 가능하다. [[ETL]] 없이는 산재된 데이터를 신뢰할 수 있는 데이터 자산으로 변환할 수 없다.

## 비유로 이해하기

[[ETL]]은 마치 여러 식재료 공급처에서 재료를 받아(Extract) 씻고 다듬어(Transform) 냉장고에 정리정돈해서 보관(Load)하는 것처럼, 여러 소스의 데이터를 수집·처리·저장하는 과정이다.

## 실제 사용 예시

전자상거래 회사가 온라인 거래 DB, 물류 시스템, CRM에서 고객 주문 정보를 매일 밤 자동으로 추출하여 중복을 제거하고 표준 형식으로 변환한 후 데이터 웨어하우스에 적재하는 배치 작업을 구성할 때 [[ETL]] 파이프라인을 사용한다.

# Archive

## 정의

아카이브(archive)는 머신러닝 모델의 학습 과정에서 생성된 체크포인트(checkpoint), 로그, 가중치 등을 장기적으로 보관하는 저장소를 의미한다. 모델의 버전 관리(version control)와 재현성(reproducibility) 확보를 위해 필수적이다.

## 왜 중요한가

실무에서 모델 성능 비교, 오류 원인 분석, 이전 버전 복구 등이 필요할 때 아카이브된 데이터 없이는 작업이 불가능하다. 특히 장시간 학습된 모델의 경우 중간 체크포인트를 보관하는 것이 학습 실패 시 시간 손실을 막아준다.

## 비유로 이해하기

박물관이 역사적 유물을 보관하듯이, 머신러닝 프로젝트에서 아카이브는 모델의 "성장 과정"을 기록하는 박물관 역할을 한다.

## 실제 사용 예시

```
모델 학습 중 100 epoch마다 아카이브에 가중치와 메트릭스를 저장하여, 
나중에 최고 성능을 낸 epoch의 모델을 복구할 수 있다.
```

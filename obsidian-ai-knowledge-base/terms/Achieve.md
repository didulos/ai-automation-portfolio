# Achieve

## 정의
머신러닝 모델이 주어진 작업(task)에서 목표 성능을 달성하거나 특정 메트릭(metric)에서 원하는 수준의 결과를 얻는 것을 의미한다. 일반적으로 [[Training]](훈련) 과정을 통해 모델의 가중치(weight)를 최적화하여 달성된다.

## 왜 중요한가
실무에서 모델 성능이 비즈니스 요구사항을 충족하는지 평가하는 핵심 지표이다. 목표 달성 여부에 따라 모델 배포(deployment) 여부와 추가 최적화 필요성이 결정된다.

## 관련 개념
- **상위 개념**: [[Model-Evaluation]](모델 평가)
- **하위 개념**: 
  - [[Convergence]](수렴) - 훈련 과정에서 손실함수가 최소값에 도달하는 상태
  - [[Performance-Threshold]](성능 임계값) - 달성해야 할 구체적인 메트릭 목표
- **연관 개념**: 
  - [[Hyperparameter-Tuning]](하이퍼파라미터 튜닝)
  - [[Cross-Validation]](교차 검증)

## 비유로 이해하기
학생이 시험에서 목표 점수(예: 90점 이상)를 "달성"하는 것처럼, 모델이 훈련을 통해 정확도 95% 이상이라는 목표를 "달성"하는 과정이다.

## 실제 사용 예시
"이 분류 모델이 테스트 데이터(test data)에서 F1-score 0.85를 달성했다." 
"[[RMSE]](평균제곱근오차)를 0.5 이하로 달성하기 위해 [[Learning-Rate]](학습률)를 조정했다."

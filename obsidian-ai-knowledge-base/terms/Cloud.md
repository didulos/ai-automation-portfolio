# Cloud

## 정의
클라우드(cloud)는 인터넷을 통해 컴퓨팅 자원(computing resources)과 저장소(storage)를 원격으로 제공받는 기술 및 서비스 모델이다. 사용자는 물리적 인프라를 소유하지 않고도 필요한 만큼만 사용하고 비용을 지불할 수 있다.

## 왜 중요한가
AI/ML 프로젝트에서 대규모 [[GPU]], 대용량 데이터 저장, 분산 처리 환경을 경제적으로 활용할 수 있게 해준다. 특히 [[AI-Terms/01-Terms/Deep-Learning]] 모델 훈련(training)과 [[AI-Terms/01-Terms/MLOps]] 운영에서 필수적인 인프라이다.

## 관련 개념
- **상위 개념**: [[Infrastructure-as-a-Service]](인프라 서비스화)
- **하위 개념**:
  - [[Cloud-Computing]](클라우드 컴퓨팅): 클라우드 위에서 실제 계산 작업을 수행하는 것
  - [[Data-Center]](데이터 센터): 클라우드 자원을 물리적으로 관리하는 시설
- **연관 개념**:
  - [[Distributed-Computing]](분산 컴퓨팅): 클라우드에서 여러 노드(node)로 작업을 분담
  - [[Scalability]](확장성): 클라우드의 핵심 장점으로 필요에 따라 자원을 동적으로 확장

## 비유로 이해하기
클라우드는 '전기 회사'와 같다. 예전엔 각 가정이 발전기를 따로 소유했듯이, 과거엔 회사가 서버를 직접 구매했다. 이제 전기를 필요한 만큼 구독하듯이, 클라우드도 필요한 컴퓨팅 자원을 구독하는 방식이다.

## 실제 사용 예시
- AWS [[EC2]](탄력적 컴퓨팅), Google Cloud [[Vertex-AI]](머신러닝 플랫폼)에서 모델을 훈련하고 배포(deployment)한다.
- Azure의 [[Storage-Account]](저장소 계정)에 [[Dataset]](데이터셋)을 저장하고 여러 분석가가 동시에 접근한다.

# Neural-Network

## 정의

신경망(Neural Network)은 생물학적 뇌의 뉴런 구조에서 영감을 받아 만든 계산 모델로, 입력층(input layer), 은닉층(hidden layer), 출력층(output layer)으로 구성된 상호 연결된 노드들의 집합이다. 각 노드는 가중치(weight)와 편향(bias)을 통해 입력 데이터를 비선형 변환하여 복잡한 패턴을 학습할 수 있다.

## 왜 중요한가

신경망은 [[AI-Terms/01-Terms/Deep-Learning]](심층학습)의 기반이 되는 핵심 구조로, 이미지 인식, 자연어 처리, 음성 인식 등 현대 AI의 대부분의 성공 사례를 가능하게 했다. 충분한 데이터와 계산 자원이 있으면 거의 모든 종류의 패턴을 근사(approximation)할 수 있다는 특성 때문에 실무에서 가장 널리 활용되는 알고리즘이다.

## 비유로 이해하기

신경망은 마치 학생이 여러 선생님(뉴런)에게 다양한 관점의 설명을 받아 종합적으로 이해하고, 그 경험을 통해 새로운 문제를 풀 수 있게 되는 과정과 비슷하다. 각 선생님은 자신의 지식(가중치)을 바탕으로 조언(활성화)을 제공하고, 학생은 이를 모두 종합해 최종 답변을 내린다.

## 실제 사용 예시

```
이미지 분류 모델: 입력층(픽셀값) → 3개의 은닉층(특징 추출) 
→ 출력층(고양이/개/새 확률)

자연어 처리: [[Embedding]](임베딩) → [[LSTM]] 신경망 → 문장 분류
```

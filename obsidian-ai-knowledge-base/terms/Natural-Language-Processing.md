# Natural Language Processing

## 정의

자연언어처리(Natural Language Processing, NLP)는 인간이 사용하는 언어를 컴퓨터가 이해하고 처리할 수 있도록 하는 [[AI]](인공지능) 분야이다. 텍스트나 음성 데이터에서 의미를 추출하고 패턴을 학습하여 다양한 언어 기반 작업을 수행한다.

## 왜 중요한가

현대의 거의 모든 언어 기반 서비스([[LLM]] 챗봇, 기계 번역, 감정 분석 등)의 기반 기술이므로, AI/ML 실무에서 필수 역량이다. 비정형 텍스트 데이터로부터 비즈니스 인사이트를 도출하는 핵심 수단이다.

## 관련 개념

- **상위 개념**: [[AI]](인공지능)
- **하위 개념**: 
  - [[Tokenization]](토큰화)
  - [[Named-Entity-Recognition]](개체명 인식)
- **연관 개념**: 
  - [[Word-Embedding]](단어 임베딩)
  - [[Transformer]](트랜스포머)

## 비유로 이해하기

자연언어처리는 외국인이 낯선 언어를 배우는 과정과 같다. 처음에는 단어 하나하나를 분석하고([[tokenization]]), 단어들 간의 관계를 학습한 후([[embedding]]), 문법과 의미를 이해하게 되는 것처럼, NLP도 텍스트를 점진적으로 분석하고 이해한다.

## 실제 사용 예시

- 고객 리뷰 데이터에서 감정 분석([[Sentiment-Analysis]])을 수행하여 제품 평가 자동화
- ChatGPT 같은 [[LLM]]을 활용한 질문-답변 시스템 구축

# Preprocessing

## 정의
전처리는 원본 데이터를 [[AI-Terms/01-Terms/Machine-Learning]](머신러닝) 모델이 학습하기에 적합한 형태로 변환하는 과정이다. 결측치 처리, 이상치 제거, 정규화(normalization) 등의 작업을 포함한다. 데이터의 품질과 일관성을 확보하여 모델 성능을 크게 좌우한다.

## 왜 중요한가
원본 데이터는 노이즈(noise), 결측치, 불균형 등의 문제를 포함하는 경우가 대부분이며, 이를 정제하지 않으면 모델의 정확도와 신뢰성이 급격히 떨어진다. 실무에서는 전체 데이터 작업 시간의 60~80%가 전처리에 할당될 정도로 중요한 단계다.

## 비유로 이해하기
요리 전에 재료를 손질하는 과정과 같다. 야채의 흙을 씻어내고, 썩은 부분을 제거하고, 먹기 좋은 크기로 자르는 작업이 전처리이다. 이 과정을 제대로 하지 않으면 아무리 좋은 요리법(모델)도 맛있는 음식(정확한 결과)을 만들 수 없다.

## 실제 사용 예시
고객 거래 데이터의 나이 필드에 음수나 200세 같은 이상치가 있다면 제거하고, 일부 고객의 전화번호가 비어있다면 대체하는 작업. 또는 0~1000만 원대의 수입을 0~1 범위로 정규화하여 모델의 수렴 속도를 높이는 작업이다.

# Image-Classification

## 정의
이미지 분류는 입력된 이미지를 미리 정의된 카테고리(클래스) 중 하나로 할당하는 [[Computer-Vision]](컴퓨터 비전) 작업이다. [[AI-Terms/01-Terms/Deep-Learning]](딥러닝) 기반의 [[CNN]](합성곱 신경망)이 이 작업에서 가장 널리 사용되며, 이미지의 시각적 특징을 학습하여 분류 성능을 높인다.

## 왜 중요한가
의료 진단, 자동차 제조 품질 검사, 소셜 미디어 콘텐츠 필터링 등 실제 산업 전반에서 자동화와 의사결정의 핵심 기술로 활용된다. 올바른 분류 모델 선택과 최적화는 비즈니스 효율성과 정확성을 크게 좌우한다.

## 관련 개념

**상위 개념:**
- [[Computer-Vision]](컴퓨터 비전)

**하위 개념:**
- [[Binary-Classification]](이진 분류): 2개 클래스로만 분류하는 경우
- [[Multi-Class-Classification]](다중 클래스 분류): 3개 이상의 클래스로 분류하는 경우

**연관 개념:**
- [[Transfer-Learning]](전이 학습): 사전 학습된 모델([[ImageNet]](이미지넷)을 기반으로)을 활용하여 새로운 이미지 분류 작업에 적용
- [[Data-Augmentation]](데이터 증강): 학습 데이터 부족 시 이미지를 회전, 확대 등으로 변형하여 모델 성능 향상

## 비유로 이해하기
우편물 분류원이 편지봉투들을 보면서 주소를 확인해 각 지역별로 분류하는 것처럼, 이미지 분류 모델도 이미지의 특징을 학습하여 자동으로 올바른 카테고리에 배치한다.

## 실제 사용 예시
의료 기관에서 X-ray 이미지를 정상/비정상으로 분류하거나, 전자상거래 플랫폼에서 사용자가 업로드한 상품 사진을 패션/전자제품/식품 등으로 자동 분류할 때 사용된다.

# Machine Learning

## 정의
데이터로부터 패턴을 자동으로 학습하여 예측이나 의사결정을 수행하는 AI(인공지능) 기술이다. 명시적으로 프로그래밍하지 않고도 경험을 통해 성능을 개선하는 알고리즘들의 집합이다.

## 왜 중요한가
대규모 데이터에서 인간이 발견하기 어려운 패턴을 자동으로 찾아낼 수 있어 예측 분석, 추천 시스템, 이상 탐지 등 실무 문제 해결의 핵심 기술이다. 현대 데이터 기반 비즈니스의 필수 역량이다.

## 비유로 이해하기
아이가 많은 사진을 보면서 "이건 개고, 이건 고양이다"라고 배우는 것처럼, 머신러닝은 컴퓨터가 대량의 데이터 예시를 통해 특정 패턴을 스스로 학습하고 새로운 데이터에 적용하는 과정이다.

## 실제 사용 예시
이메일 스팸 필터는 과거 스팸/정상 이메일 데이터로 학습하여 새 이메일이 스팸인지 판별한다. 넷플릭스의 추천 알고리즘은 사용자의 시청 패턴을 학습하여 맞춤형 콘텐츠를 제시한다.

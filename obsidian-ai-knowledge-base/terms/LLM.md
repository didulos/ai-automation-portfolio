# LLM

## 정의
[[LLM]](대규모 언어 모델)은 수십억 개 이상의 매개변수를 가진 [[AI-Terms/01-Terms/Deep-Learning]](심층학습) 모델로, 방대한 텍스트 데이터로 사전 학습되어 자연언어 이해 및 생성 능력을 갖춘다. [[Transformer]](트랜스포머) 아키텍처 기반으로 다음 토큰 예측(next token prediction) 방식으로 학습되며, 지시(instruction) 수행, 질문 응답, 텍스트 생성 등 다양한 작업을 한 번의 모델으로 처리할 수 있다.

## 왜 중요한가
현대 AI의 기반 모델(foundation model)로서 [[Fine-Tuning]](미세조정) 없이도 즉시 실무에 적용 가능하며, [[AI-Terms/01-Terms/NLP]](자연언어처리)의 거의 모든 작업에서 최고 수준의 성능을 달성한다. 기업에서 생성형 AI 서비스의 핵심 엔진으로 활용되고 있어 필수적으로 이해해야 할 기술이다.

## 관련 개념
- **상위 개념**: [[Foundation-Model]](기반 모델)
- **하위 개념**: 
  - [[GPT]](생성 사전학습 트랜스포머)
  - [[BERT]](양방향 인코더 표현)
- **연관 개념**: 
  - [[Prompt-Engineering]](프롬프트 엔지니어링)
  - [[AI-Terms/01-Terms/RAG]](검색증강생성)

## 비유로 이해하기
[[LLM]]은 방대한 책을 읽고 패턴을 학습한 사람과 같다. 특정 주제에 대해 질문하면, 읽었던 내용의 패턴을 바탕으로 자연스러운 답변을 생성해낸다. 책을 더 많이 읽을수록(더 큰 데이터로 학습할수록) 더 정교한 답변을 제공할 수 있다.

## 실제 사용 예시
ChatGPT, Claude, Gemini 같은 챗봇 서비스가 [[LLM]]을 기반으로 사용자 질문에 자연스러운 답변을 생성한다. 기업에서는 [[AI-Terms/01-Terms/RAG]]와 결합하여 내부 문서 기반 고객 상담 시스템을 구축하거나, [[Fine-Tuning]]을 통해 특정 도메인 작업에 최적화한 [[LLM]]을 구축한다.

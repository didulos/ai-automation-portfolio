# Prompting

## 정의

[[LLM]](대규모 언어 모델)이나 [[AI]] 모델에게 원하는 작업을 수행하도록 지시하는 입력 텍스트를 설계하고 최적화하는 기술이다. 효과적인 프롬프트는 모델의 성능과 출력 품질을 크게 좌우한다.

## 왜 중요한가

[[LLM]]의 능력을 제대로 활용하기 위해 필수적인 기술로, 같은 모델이라도 프롬프트 작성 방식에 따라 정확도와 유용성이 극적으로 달라진다. 코드 수정 없이 모델 성능을 향상시킬 수 있는 가장 실용적인 방법이다.

## 관련 개념

- **상위 개념**: [[AI-Terms/01-Terms/NLP]](자연언어처리)
- **하위 개념**: 
  - [[Few-Shot-Prompting]](소수샷 프롬프팅) - 모델에게 몇 가지 예시를 제공하여 학습
  - [[Chain-of-Thought]](사고의 연쇄) - 단계적 추론 과정을 거치도록 유도
- **연관 개념**:
  - [[Prompt-Engineering]](프롬프트 엔지니어링) - 프롬프트 설계를 체계적으로 개선하는 분야
  - [[Context-Window]](컨텍스트 윈도우) - 모델이 처리할 수 있는 입력 길이 제한

## 비유로 이해하기

요리사에게 완성된 요리를 지시하는 방식이 좋을수록 원하는 맛의 음식이 나오는 것처럼, [[LLM]]에게 명확하고 구체적인 지시(프롬프트)를 줄수록 더 정확한 답변을 얻을 수 있다.

## 실제 사용 예시

**효과적인 프롬프트**: "당신은 경험 10년의 데이터 분석가입니다. 다음 CSV 데이터의 이상치를 찾고, 각 이상치마다 원인을 제시해주세요."

**부정확한 프롬프트**: "데이터 분석해줘"

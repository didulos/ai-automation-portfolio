## 1. 제품 개요

| 항목 | 내용 |
|------|------|
| **제품명** | ArumDri 스마트스토어 CS봇 |
| **타겟** | 네이버 스마트스토어 운영 소규모 셀러 (1~3인 운영) |

## 2. 핵심 기능

| # | 기능 | 설명 |
|---|------|------|
| F1 | FAQ 자동 응답 | 배송, 교환/반품, 재입고, 영업시간, 결제 등 반복 문의 자동 처리 |
| F2 | 스토어별 맞춤 정보 반영 | 배송업체, 배송 소요일, 교환 정책 등 셀러별 변수 자동 치환 |
| F3 | 폴백 처리 | 답변 불가 문의 → "담당자에게 전달하겠습니다" 응답 + 셀러 알림 |
| F4 | 대화 이력 저장 및 조회 | Airtable에 모든 대화 저장, 셀러가 조회 가능 |
| F5 | 듀얼 채널 지원 | 카카오톡 채널 챗봇 + 웹 채팅 위젯 (스토어 임베드) |

### S01. 배송 조회

**고객 메시지 예시**
1. "제 주문 언제 와요?"
2. "택배 운송장 번호 알려주세요"
3. "주문한 지 3일 됐는데 아직 안 왔어요"

**봇 응답 템플릿**
```
안녕하세요! {{StoreName}} 입니다 😊

배송 조회는 아래 방법으로 확인하실 수 있어요:

📦 배송사: {{ShippingCarrier}}
🔗 조회 링크: {{TrackingUrl}}

운송장 번호는 배송 시작 후 발송되는 카카오 알림톡에서 확인하실 수 있어요.
추가 문의는 언제든 말씀해 주세요!
```

**필요한 스토어 변수**
| 변수 | 설명 |
|------|------|
| `{{StoreName}}` | 스토어 이름 |
| `{{ShippingCarrier}}` | 주로 사용하는 택배사명 |
| `{{TrackingUrl}}` | 택배사 조회 링크 |

### S02. 배송 소요일 문의

**고객 메시지 예시**
1. "주문하면 며칠 만에 와요?"
2. "오늘 주문하면 이번 주 내로 받을 수 있나요?"
3. "배송 얼마나 걸려요?"

**봇 응답 템플릿**
```
배송 안내드릴게요! 🚚

📅 출고 기준: 결제 완료 후 {{ShipOutDays}}영업일 이내 출고
🏠 수령 기준: 출고 후 {{ShippingDays}}영업일 내 수령

※ {{CutoffTime}} 이전 결제 건은 당일 출고 처리됩니다.
※ 주말·공휴일 제외

궁금한 점 있으시면 편하게 문의해 주세요!
```

**필요한 스토어 변수**
| 변수 | 설명 |
|------|------|
| `{{ShipOutDays}}` | 출고 소요 영업일 |
| `{{ShippingDays}}` | 배송 소요 영업일 |
| `{{CutoffTime}}` | 당일 출고 마감 시간 (예: 오후 2시) |

### S03. 교환 요청

**고객 메시지 예시**
1. "사이즈를 잘못 주문했어요. 교환 가능한가요?"
2. "상품이 불량인데 교환해 주세요"
3. "색깔이 사진이랑 달라요. 바꿀 수 있나요?"

**봇 응답 템플릿**
```
교환 문의 주셨군요!

{{StoreName}} 교환 정책이에요:
📅 교환 가능 기간: 수령 후 {{ExchangePeriod}}일 이내
✅ 교환 가능: 상품 불량, 오배송, 미사용·미개봉 상태
❌ 교환 불가: 사용 후 단순 변심, 고객 부주의 훼손

교환을 원하시면 아래 정보를 보내주세요:
1. 주문번호
2. 교환 사유
3. 원하시는 옵션 (사이즈/색상 등)

확인 후 빠르게 안내드릴게요!
```

**필요한 스토어 변수**
| 변수 | 설명 |
|------|------|
| `{{StoreName}}` | 스토어 이름 |
| `{{ExchangePeriod}}` | 교환 가능 기간 (일) |

---

### S04. 반품/환불 요청

**고객 메시지 예시**
1. "마음에 안 들어서 반품하고 싶어요"
2. "환불은 어떻게 신청해요?"
3. "불량품 받았는데 전액 환불 가능한가요?"

**봇 응답 템플릿**
```
반품/환불 안내드릴게요.

📅 반품 가능 기간: 수령 후 {{ReturnPeriod}}일 이내
🚚 반품 배송비:
  - 단순 변심: {{ReturnShippingCost}}원 (고객 부담)
  - 상품 불량·오배송: 무료 (판매자 부담)

반품 신청 방법:
네이버 앱 → 주문 내역 → 반품 신청

환불은 반품 상품 확인 후 {{RefundDays}}영업일 이내 처리됩니다.
```

**필요한 스토어 변수**
| 변수 | 설명 |
|------|------|
| `{{ReturnPeriod}}` | 반품 가능 기간 (일) |
| `{{ReturnShippingCost}}` | 단순 변심 반품 배송비 (원) |
| `{{RefundDays}}` | 환불 처리 소요 영업일 |

### S05. 주문 취소

**고객 메시지 예시**
1. "방금 주문했는데 취소할 수 있나요?"
2. "아직 발송 전이면 취소해 주세요"
3. "주문 취소 어떻게 해요?"

**봇 응답 템플릿**
```
주문 취소 안내드릴게요!

출고 전이라면 직접 취소 가능해요:
👉 네이버 앱 → 주문 내역 → 취소 신청

⚠️ 이미 출고된 경우 취소가 불가하며,
배송 완료 후 반품으로 진행해 주셔야 해요.

취소 후 환불은 결제 수단에 따라 {{RefundDays}}영업일 내 처리됩니다.
혹시 취소가 안 되신다면 주문번호를 알려주시면 확인해 드릴게요!
```

**필요한 스토어 변수**
| 변수 | 설명 |
|------|------|
| `{{RefundDays}}` | 환불 처리 소요 영업일 |

재입고 확정 시 알림톡으로 바로 안내드릴게요.

```
{{StoreName}} 운영 안내예요 🕐

📅 운영일: {{BusinessDays}}

**필요한 스토어 변수**

| `{{HolidayInfo}}` | 휴무 안내 |

### S08. 결제 수단 / 세금계산서 문의

**고객 메시지 예시**

```
결제 관련 안내드릴게요 💳

사용 가능한 결제 수단:
{{PaymentMethods}}

{{#if TaxInvoiceAvailable}}
📄 세금계산서 발행 가능합니다.

| 변수 | 설명 |

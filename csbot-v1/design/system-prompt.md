## 사용 방법

n8n 워크플로우에서 Claude API 호출 시 아래 **System Prompt**와 **User Prompt**를 분리하여 전달한다.

- `{{store_name}}` — Airtable Stores.StoreName
- `{{store_info}}` — Stores 테이블 주요 필드를 key: value 형식으로 직렬화한 텍스트
- `{{faq_data}}` — FAQs 테이블에서 해당 스토어의 활성 FAQ 전체를 JSON 직렬화한 텍스트
- `{{conversation_history}}` — 최근 5턴 대화 이력 (role/content 형식)
- `{{customer_message}}` — 고객이 방금 보낸 메시지 원문

## System Prompt

```
당신은 {{store_name}}의 고객 서비스 상담원입니다.

## 역할
고객의 문의에 FAQ 데이터를 기반으로 정확하고 친절하게 답변합니다.
절대 FAQ 데이터 외의 내용을 추측하거나 임의로 답변하지 않습니다.

## 말투 규칙
- 항상 존댓말을 사용합니다.
- 따뜻하고 전문적인 어조를 유지합니다.
- 이모지는 응답당 최대 1개만 사용합니다.
- 응답은 3문장 이내로 간결하게 작성합니다.

## 스토어 기본 정보
{{store_info}}

## FAQ 데이터
아래 FAQ 목록만을 근거로 답변합니다. 이 목록에 없는 내용은 절대 답변하지 않습니다.

{{faq_data}}

## 답변 규칙

1. **FAQ 매칭 시**: FAQ의 AnswerTemplate을 기반으로 스토어 정보 변수를 채워 자연스럽게 답변합니다.
2. **FAQ 미매칭 시**: 아래 폴백 문구를 그대로 사용합니다.
   > "문의 주신 내용을 담당자에게 전달하겠습니다. 영업시간 내 순차적으로 확인 후 연락드리겠습니다."
3. **복합 문의 시**: 가장 핵심적인 질문 하나에만 답변하고, 나머지는 추가 문의를 유도합니다.

## 절대 금지 사항
- 가격 할인이나 추가 혜택 약속
- 환불·교환의 즉시 처리 약속
- 배송 날짜를 구체적 날짜로 확정하여 약속
- 주민등록번호, 계좌번호 등 민감한 개인정보 요청
- FAQ에 없는 정책·정보를 추측하여 안내
- 타 쇼핑몰이나 경쟁사 언급
```

교환 가능 기간: 수령 후 7일

단순 변심 반품 배송비: 3,000원

영업시간: 09:00~18:00 (월~금)

  },
  {
    "category": "교환",

### `{{conversation_history}}` 예시
```
[Customer]: 배송 얼마나 걸려요?
[Bot]: 결제 완료 후 1영업일 이내 출고되며, 출고 후 3영업일 내 수령 가능합니다.
[Customer]: 오늘 주문하면 이번 주 금요일까지 받을 수 있나요?
```

## 토큰 비용 추정

| 구성 요소 | 예상 토큰 | 비고 |
|-----------|-----------|------|
| System Prompt (고정) | ~300 | 역할·규칙 |
| `{{store_info}}` | ~200 | 스토어 설정 직렬화 |
| `{{faq_data}}` (10개) | ~800 | FAQ 전체 주입 |
| `{{conversation_history}}` (5턴) | ~300 | 대화 이력 |
| `{{customer_message}}` | ~50 | 고객 메시지 |
| **Input 합계** | **~1,650** | |
| **Output** | **~150** | 3문장 이내 응답 |
| **1회 호출 총합** | **~1,800 토큰** | |

> **claude-sonnet-4-6 실제 비용 계산** (환율 $1 = ₩1,300 기준)
> - Input: 1,650 토큰 × $3/MTok × 1,300 ≈ **₩6.4**
> - Output: 150 토큰 × $15/MTok × 1,300 ≈ **₩2.9**
> - **1회당 합계: 약 ₩9.3**
>
> **일 5,000원 기준 처리 가능 건수: 약 540건/일**
> 일 1,000건 초과 시 약 ₩9,300/일 → PRD 제약 조건 재검토 필요
> 초기 소량 운영(~500건/일) 또는 haiku 모델로 비용 절감 고려
>
> ※ ~2026-03-27까지 토큰 2배 프로모션 적용 시: 1회당 약 ₩4.7, 일 5,000원 기준 ~1,060건/일
